💡 认知冲突 · 为什么一样多?

🤔 三位数回文数:90 个
🤔 四位数回文数:90 个

竟然一样多!为什么?

关键发现:三位数有 2 个"自由位"(百位 a 和十位 b),四位数也有 2 个"自由位"(千位 a 和百位 b)。
回文数的个数不取决于位数,而取决于"自由位"的个数!

🧬 DNA 回文密码 —— 生命的对称

DNA 是双链结构,A 与 T 配对,G 与 C 配对。当上链从左往右读 = 下链从右往左读时, 就形成了"回文序列"—— 它是基因工程、疫苗研发、遗传病治疗的关键标志。

💻 计算机中的回文判定算法

让计算机判断回文数的核心代码(JavaScript):

// 判断一个数是不是回文数 —— 就是我们这节课学的方法! function isPalindrome(n) { const s = n.toString(); // 把数变成字符串 const r = s.split('').reverse().join(''); // 翻转 return s === r; // 对应位比较 } isPalindrome(121); // → true ✓ isPalindrome(123); // → false ✗ isPalindrome(12321); // → true ✓
🎯 关键洞察:这段代码的核心是"对应位比较" —— 和我们这节课判断回文数的方法完全一样! 回文判定是程序员的经典入门题,也是数据压缩、密码学、错误检测的基础技术。

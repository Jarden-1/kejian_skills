💻 计算机中的回文算法

// 判断一个数是不是回文数
function isPalindrome(n) {
const s = n.toString();
const r = s.split('').reverse().join('');
return s === r; // 对应位比较
}

核心:对应位比较 —— 和我们今天学的方法完全一样!
回文判定 → 数据压缩、密码学、错误检测的基础
